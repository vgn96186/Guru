Design Notes

The hub is a single, calm dock (not a busy FAB). It telegraphs the 4 core “today behaviors” and expands into a tight 2×2 grid, optimized for thumb reach and tablet ergonomics.

Closed state: summary + mood signal (signature bar) One decision
Open state: 4 actions, each with a distinct color memory 2×2 grid
Motion: scale + lift, minimal jitter, focus-ring for keyboard Accessible
Visual language: glass + neon clinical accents + subtle grain Android vibe
