Guru Dialog System

Linear-themed glassmorphic dialogs — all variants

default — info / general
DEFAULT
Clear AI Cache?
All cached content cards will be regenerated fresh on next use.
success — operation completed
SUCCESS
Backup Complete
Your study data has been backed up to Google Drive successfully.
warning — caution needed
WARNING
No Valid Transcripts
Selected transcripts are too short to process. Choose longer recordings.
error — something failed
ERROR
Something went wrong
Failed to expand node: Network request timed out. Please try again.
focus — confirmation (confirm)
FOCUS
Run Auto-Backup Now?
This will create an automatic backup regardless of your frequency setting.
destructive — dangerous action
DESTRUCTIVE
Reset all progress?
This clears all topic progress, XP, streaks, and daily logs. This cannot be undone.

Migration Examples

// ❌ Before (native Alert)
Alert.alert('Reset all progress?', 'This cannot be undone.', [
  { text: 'Cancel', style: 'cancel' },
  { text: 'Reset', style: 'destructive', onPress: resetProgress },
]);

// ✅ After (themed dialog)
const ok = await confirmDestructive(
  'Reset all progress?',
  'This clears all topic progress, XP, streaks, and daily logs. This cannot be undone.'
);
if (!ok) return;
resetStudyProgress();


// ❌ Before
Alert.alert('Connected', 'Google Drive is now active.');

// ✅ After
showSuccess('Connected', 'Google Drive is now active.');


// ❌ Before
Alert.alert('AI Error', err.message);

// ✅ After
showError(err, 'Failed to expand node');


// ❌ Before — complex 3-button alert
Alert.alert('Title', 'Message', [
  { text: 'Cancel' },
  { text: 'Backup', onPress: doBackup },
  { text: 'Skip', onPress: skip },
]);

// ✅ After — full showDialog
await showDialog({
  title: 'Title',
  message: 'Message',
  variant: 'focus',
  actions: [
    { id: 'cancel', label: 'Cancel', variant: 'secondary' },
    { id: 'backup', label: 'Backup', variant: 'primary', onPress: doBackup },
    { id: 'skip', label: 'Skip', variant: 'secondary', onPress: skip },
  ],
  allowDismiss: true,
});
